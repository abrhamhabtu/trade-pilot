## 💡 Why Enterprise Edition?

We believe in open source, but building great software requires sustainable funding.

**Free Core (AGPL-3.0):**
- ✅ Full trade import (CSV, Excel)
- ✅ Multi-account support
- ✅ Calendar & performance charts
- ✅ Playbooks & strategy tracking
- ✅ Journal entries
- ✅ Local data storage
- ✅ All current features

**Paid Enterprise:**
- 💰 Cloud sync & backup
- 💰 Team collaboration
- 💰 Advanced AI features
- 💰 Broker integrations
- 💰 Priority support
